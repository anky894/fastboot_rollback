- Extract stock images in a directory and cd into it.
- `git clone https://github.com/anky894/fastboot_rollback.git flash && cd flash `
- Reboot to fastbootD
- `./flash.sh`
- If everything flashed successfully, do `fastboot reboot bootloader`
- `fastboot -w`
- `fastboot reboot`
